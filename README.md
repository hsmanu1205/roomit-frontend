# RoomIt Frontend

Frontend for RoomIt Meeting Room Booking System built using Next.js, React, Tailwind CSS and Axios.

---

## Live Demo

🔗 Live Demo: https://roomit-frontend-alpha.vercel.app/

---

## Features

- View Available Rooms
- Room Availability Grid
- Dynamic Slot Updates
- Create Booking
- Search Bookings by Email
- Cancel Bookings
- Responsive UI
- Modern Dashboard Interface

---

## Tech Stack

- Next.js 15
- React
- Tailwind CSS
- Axios

---

## Pages

### Home

```bash
/
```

Shows:

- All Meeting Rooms
- Room Details
- Availability Access

### Room Availability

```bash
/rooms/[id]
```

Features:

- Date Selection
- Real-Time Availability
- Available/Unavailable Slots

### Create Booking

```bash
/book/[id]
```

Features:

- Meeting Title
- Name
- Email
- Booking Creation

### My Bookings

```bash
/bookings
```

Features:

- Search by Email
- View Booking History
- Cancel Booking

## Installation

Clone Repository

```bash
git clone YOUR_FRONTEND_REPOSITORY_URL
```

Install Dependencies

```bash
npm install
```

Create Environment File

```env
NEXT_PUBLIC_API_URL=http://localhost:5000/api
```

Run Development Server

```bash
npm run dev
```

Frontend runs at:

```bash
http://localhost:3000
```

---

## Build

```bash
npm run build
```

## Production

```bash
npm start
```
